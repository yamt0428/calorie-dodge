# カロリーセーブ (Calorie Dodge)

誘惑に負けずに高カロリー食品を買わなかった際に、そのカロリーを記録することで、健康意識の向上とモチベーション維持を図るアプリです。

## 主な機能

### 記録機能
- カロリーの記録（必須）と商品名/メモ（任意）の入力
- 簡易入力UI（プリセット付き）
- 過去の記録の編集・削除

### ホーム画面
- 今日の回避カロリー（大きく表示）
- 累計回避カロリー
- 連続記録日数
- GitHubスタイルのヒートマップカレンダー

### 統計・グラフ機能
- 日別・週別・月別の推移グラフ
- 総記録回数、最も記録の多い時間帯
- 平均回避カロリー、1日の最大回避カロリー

### 目標設定機能
- 期間ベースの目標（週間/月間で○○kcal回避）
- 連続記録の目標（連続○日記録）
- 進捗状況をパーセンテージで表示

### バッジ機能
累計カロリー、連続記録日数、記録回数でバッジを獲得できます。

- **累計カロリーバッジ**: 1,000 / 5,000 / 10,000 / 30,000 / 50,000 kcal
- **連続記録日数バッジ**: 3 / 7 / 14 / 30 / 100 日
- **記録回数バッジ**: 10 / 50 / 100 / 300 / 500 回

### SNSシェア機能
- 週間達成報告のシェア
- 累計成果のシェア
- データのエクスポート

## 技術スタック

- **フレームワーク**: Flutter
- **対応プラットフォーム**: iOS / Android 両対応
- **データ保存**: Hive（ローカルストレージ）
- **状態管理**: Provider
- **グラフ描画**: fl_chart

## 開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/yamt0428/calorie-dodge.git
cd calorie-dodge/calorie_dodge

# 依存関係をインストール
flutter pub get

# Hiveアダプタを生成
flutter pub run build_runner build

# アプリを実行
flutter run
```
